Key Design Decisions

Greeting

Time-aware ("Good morning/afternoon/evening") + today's session count. Returning after missed days shows "Welcome back 💛" — never guilt.

Task Input

Optional, never required. Naming your task reduces ADHD paralysis ("What should I work on??" → just type something). Saved in session history.

Duration Picker

5 / 15 / 25 / Custom — the 5-minute option is crucial for ADHD. "I can do 5 minutes" breaks the starting barrier. Many users discover they keep going past 5. The ⚙️ button opens a custom minutes input.

Streak Display

Visible but not dominant. Top-left, small. Motivating without creating pressure. After a broken streak: "Starting fresh 🌱" not "Streak: 0".

Dashboard Link

📊 icon top-right. Not prominent — this screen is about action, not analysis.

Keyboard Shortcut

Spacebar to start. Zero friction for power users.

🧠 ADHD Design Principle

Everything on this screen exists to answer one question: "Can I just start?" Yes. One tap. No setup required. Task name optional. Duration pre-selected. The breathing ritual is on the NEXT screen, not here. Zero gates between opening the app and beginning.
